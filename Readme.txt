ЗАПУСТИТЬ МОЖНО ИЗ ФАЙЛА package.json

Установите зависимости:
В командной строке выполните команду npm install для установки всех зависимостей, указанных в файле package.json.

npm install

Запустите проект в режиме разработки:
В командной строке выполните команду npm run dev. Эта команда запустит проект в режиме разработки с использованием Vite.

npm run dev
После выполнения этой команды, вы увидите вывод в консоли, и ваше приложение будет доступно по адресу http://localhost:3000 (или другому порту, если этот занят). Откройте этот URL в браузере, чтобы увидеть ваше приложение.

После чего переход на 
http://127.0.0.1:5174/

Сборка проекта:
Если вы хотите создать оптимизированную сборку вашего проекта, используйте команду npm run build.

npm run build
Это создаст оптимизированные файлы вашего проекта в папке dist.

Линтинг кода:
Если вы хотите запустить линтинг кода с использованием ESLint, выполните команду:

npm run lint
Это поможет вам выявить и исправить потенциальные проблемы в вашем коде.

Предпросмотр сборки:
Если вы хотите предпросмотреть собранное приложение перед развертыванием, используйте команду:

npm run preview
Эта команда запустит локальный сервер с предварительным просмотром собранного приложения.

После чего переход на 
http://127.0.0.1:4173/
